# League UI (tablet-first)

このフォルダを **そのままGitHubにアップ** すると動きます（静的サイト）。

## 使い方
- `index.html` を開く
- 右上「管理」から
  - リーグ名/ロゴ編集
  - リーグ追加/削除
  - シーズン切替（過去戦績保持）
  - ディビジョン追加/削除、チーム追加/削除、編集
- 「日程生成」で
  - 総当たり回数
  - Home & Away の有無
  を指定して自動生成
- 日程の「入力」からスコア入力 → 順位表に即反映（勝3/分1/負0）
- チーム名（枠つき）をタップ → チーム戦績/試合一覧/編集
- 「順位カラー編集」で順位に応じた左の細線カラーを編集

## 保存について
データはブラウザの LocalStorage に保存されます（サーバ不要）。
